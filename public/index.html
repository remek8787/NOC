Catatan engineering

Prinsip desain

  • Port API per router wajib configurable dari dashboard.
  • SNMP version / port / security profile juga wajib configurable per device.
  • Backup OLT harus modular per vendor, tidak boleh diasumsikan seragam.
  • Hosting web app bisa tetap di subdomain ini, sementara collector berat bisa jalan di VM terpisah jika dibutuhkan.
  • Target akhir: usable real untuk NOC, bukan sekadar mockup cantik.
