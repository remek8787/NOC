# NOC Web App

Blueprint + MVP dashboard shell untuk `noc.anantasatriya.my.id`.

## Scope awal
- Dashboard overview NOC
- Multi-device model (router / OLT / switch / AP)
- Integrasi future-ready: API, SNMP, CLI backup, BGP view, route lookup
- Deploy cepat ke hosting FTP

## Rencana fase
1. MVP shell dashboard live
2. Device registry + credential vault abstraction
3. Collector API/SNMP backup service
4. BGP/route explorer
5. Config backup center
6. Alerting + audit log
